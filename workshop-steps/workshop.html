Streaming Data Workshop

Ex1 :: Playground

  • Start a cluster with the service catalog
    • cd streaming-data-workshop
    ./start-openshift.sh
  • Log in into https://127.0.0.1:8443 with developer/developer
  • Click in "browse catalog"
  • In openshift catalog, choose Infinispan Ephemeral
  • Add the following configuration : 
                APPLICATION_NAME=datagrid
            NUMBER_OF_INSTANCES=3
            MANAGEMENT_USER/MANAGEMENT_PASSWORD=developer/developer
  • Click Next
  • Click Create
  • Go to My project
  • You should see datagrid deployment with 3 pods and 3 services : datagrid-hotrod, datagrid-http and datagrid-management
You can deploy de cluster easily from command line :
        > cd streaming-data-workshop
        > ./start-datagrid.sh

We are going to work in the project now, in the simple-web-application module. The main goal here is to start and deploy your first web application that will put and get a value from infinispan. This quick exercise will help you get into the Vert.x and Infinispan API.

  • In the project, simple-web-application module
  • Create your first Verticle
  • Deploy with ./deploy.sh and redeploy with ./redeploy.sh
